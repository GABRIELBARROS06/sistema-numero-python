📌 Este projeto é um sistema interativo desenvolvido em Python, executado via terminal, que permite ao usuário inserir e analisar números.  O sistema foi criado com o objetivo de praticar lógica de programação, estruturas de repetição, condicionais e manipulação de listas.

 ⚙️ Funcionalidades

- Inserção de números pelo usuário
- Identificação de números pares
- Cálculo do maior número inserido
- Cálculo da média dos números
- Menu interativo no terminal

▶️ Como executar o projeto

1. Clone o repositório:
2. git clone https://github.com/seu-usuario/sistema-numeros-python.git
3.  Acesse a pasta do projeto:
4.  cd sistema-numeros-python
5. Execute o programa:
6. python sistema_numeros.py
